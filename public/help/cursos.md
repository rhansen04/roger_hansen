# Gestão de Cursos

O módulo de Cursos permite criar e gerenciar cursos online com módulos (seções) e lições organizadas hierarquicamente.

## Estrutura

```
Curso
├── Módulo 1
│   ├── Lição 1.1
│   ├── Lição 1.2
│   └── Lição 1.3
├── Módulo 2
│   ├── Lição 2.1
│   └── Lição 2.2
└── Módulo 3
    └── ...
```

## Como Criar um Curso

### Passo 1: Acessar
No menu lateral, clique em **Cursos** (seção "Ensino").

### Passo 2: Novo Curso
Clique no botão **"+ Novo Curso"**.

### Passo 3: Preencher Informações
- **Título**: Nome do curso
- **Descrição curta**: Resumo exibido na listagem
- **Descrição completa**: Detalhes do curso
- **Imagem de capa**: Upload da imagem
- **Categoria**: Classificação do curso
- **Nível**: Iniciante, Intermediário ou Avançado
- **Instrutor**: Professor responsável
- **Preço**: Valor ou marque como gratuito

### Passo 4: Adicionar Módulos
Após criar o curso, adicione módulos (seções) para organizar o conteúdo.

### Passo 5: Adicionar Lições
Dentro de cada módulo, crie lições com conteúdo em texto, vídeo ou ambos.

## Lista de Cursos

A tabela principal mostra:
- Título e instrutor
- Quantidade de seções, lições e alunos
- Status (Ativo/Inativo)
- Preço (Gratuito ou valor)

Use a **busca rápida** para filtrar por nome ou instrutor.

## Dicas

> Organize os módulos em ordem lógica de aprendizado. É possível reordenar módulos e mover lições entre módulos.

> Cursos inativos não aparecem para os alunos, mas mantêm todo o conteúdo para reativação futura.
