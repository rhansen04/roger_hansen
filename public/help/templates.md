# Templates de Planejamento

Templates são modelos reutilizáveis que definem a estrutura dos planejamentos pedagógicos. A coordenação cria os templates e os professores os utilizam ao preencher seus planejamentos.

## Tipos de Template

- **PFI (Planejamento de Formação Individual)** — Para turmas de berçário e maternal
- **PFII (Planejamento de Formação II)** — Para turmas de pré-escola

Cada template pode ser customizado com seções e campos específicos.

## Como Criar um Template

### Passo 1: Acessar
No menu lateral, clique em **Templates Planej.** (seção "Ensino").

### Passo 2: Novo Template
Clique no botão **"+ Novo Template"**.

### Passo 3: Informações Básicas
- **Nome**: Ex.: "Planejamento PFI - Berçário II"
- **Descrição**: Breve explicação do objetivo
- **Faixa Etária**: Selecione a faixa correspondente
- **Status**: Ativo (disponível para uso) ou Inativo

### Passo 4: Adicionar Seções
Cada seção agrupa campos relacionados. Exemplos:
- "Campos de Experiência"
- "Objetivos de Aprendizagem"
- "Metodologia"
- "Avaliação"

### Passo 5: Adicionar Campos às Seções
Dentro de cada seção, adicione campos com tipos variados:
- **Texto curto**: Para respostas breves
- **Texto longo**: Para descrições detalhadas
- **Data**: Para seleção de datas
- **Seleção**: Para escolha entre opções pré-definidas
- **Checkbox**: Para marcação múltipla
- **Radio**: Para escolha única

## Gerenciando Templates

Na página de templates, cada card mostra:
- Nome e status (ativo/inativo)
- Descrição resumida
- Faixa etária
- Botões para editar ou excluir

> **Atenção:** Excluir um template não afeta planejamentos já criados com ele, mas impede a criação de novos planejamentos com esse modelo.

## Dicas

> Mantenha os templates organizados por faixa etária para facilitar a escolha dos professores.

> Teste o template criando um planejamento de exemplo antes de disponibilizá-lo para toda a equipe.
